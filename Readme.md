
# 🌍 Viagens Landing Page

Projeto desenvolvido como desafio do curso de Front-End.

## 📌 Sobre o projeto

Esta é uma landing page de viagens com foco em apresentar um destino turístico de forma simples e atrativa.

A página contém um banner principal e três seções informativas com navegação interna.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3

## 🎯 Funcionalidades

- Navegação interna entre seções
- Banner com imagem e texto sobreposto (overlay)
- Efeito hover nos links e botões
- Layout simples e responsivo
- Botões de retorno ao topo
